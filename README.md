## 📝 Short Description

> Game Roulette Monitor is a Windows-based C++ application that randomly selects a game from user-defined shortcuts, launches it, and monitors its performance during gameplay. The tool logs system usage statistics and FPS data for each gaming session, providing a lightweight performance overview per game or per match.

---

## 🚀 Core Features

* 🎲 **Random Game Selection**
  Selects a random game from user-chosen `.lnk` or `.url` shortcuts.

* 📊 **Real-Time Performance Monitoring**

  * RAM usage (average, minimum, maximum)
  * CPU usage (average)
  * GPU temperature (average, NVIDIA GPUs)

* 🎯 **FPS Tracking (PresentMon)**
  Automatically launches `PresentMon.exe` and calculates average FPS from frame time data.

* 🧵 **Multithreaded Design**
  Performance data is collected in a dedicated monitoring thread while the game runs.

* 📝 **Session Logging**
  All results are saved to a persistent log file for later review.

---

## 🎮 League of Legends Special Mode

If the game name contains **"League Of Legends"**, the application enables a multi-match workflow:

* User selects the number of matches
* Each match is monitored independently
* FPS is calculated per match
* PresentMon is restarted between matches

---

## 🖥️ How It Works

1. User selects twenty game shortcuts
2. One shortcut is chosen at random
3. The game is launched
4. User provides the game name and process PID
5. The application:

   * Starts PresentMon
   * Monitors system performance every 5 seconds
6. When the game exits:

   * FPS is calculated from CSV data
   * Results are written to `game_logs.txt`
   * Temporary files are cleaned up

## 🛠️ Technologies & APIs

* **C++ (Windows)**
* **Windows API** (process, memory, privileges)
* **NVAPI** (NVIDIA GPU temperature)
* **PresentMon** (FPS capture)
* **COM & UI Automation** (file dialog, Task Manager automation)
* **Multithreading** (`CreateThread`)

## ⚠️ Requirements

* Windows OS
* Administrator privileges (recommended)
* NVIDIA GPU (for GPU temperature monitoring)
* Windows SDK - https://learn.microsoft.com/en-us/windows/apps/windows-sdk/downloads
* Visual Studio (only to avoid errors before running the program)
* `PresentMon.exe` located in the same directory as the executable
